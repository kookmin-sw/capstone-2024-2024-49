## 🔉 프로젝트 소개

###  🍅무꾸리🍅 : 점술 채팅 상담(Prototype)
프로젝트 소개
코로나 이후 온택트 플랫폼들이 
점술 시장에도 큰 영향을 미쳤습니다. 
이에 따라 오프라인 점집보다 
온라인 점술 어플이 많이 생겨나고 
사람들은 소비하고 있습니다. 

하지만 현재의 점술 어플들은 
대부분 소비자의 니즈 충족에만 초점을 맞추고 있어 
기능들의 큰 차이점이 존재하지 않습니다.

그래서 상담사의 입장에서 
편리한 기능들을 추가하여, 
상담사들이 더 편리하게 사용할 수 있는 
점술 상담 어플을 만드는 것을 목표로 하고 있습니다
